# Тестовое задание DevOps — Effective Mobile

## Описание

Проект разворачивает простое веб-приложение с использованием Docker и Docker Compose.

Архитектура состоит из двух сервисов:
- backend — минимальный HTTP-сервер на Python
- nginx — reverse-proxy, принимает запросы от пользователя

## Схема работы:
Пользователь
     |
     v
localhost:80
     |
   nginx
     |
     v
backend:8080

## Описание сервисов

### Backend

- Реализован на Python с использованием стандартного `http.server`
- Слушает порт 8080
- Порт не пробрасывается наружу, доступен только внутри docker-сети
- При запросе на `/` возвращает текст:

---

### Nginx

- Используется официальный образ `nginx:alpine`
- Слушает порт 80
- Порт 80 проброшен на хост
- Проксирует HTTP-запросы на backend по имени сервиса
- Используется отдельный конфигурационный файл

## Запуск проекта

### Требования
- Docker
- Docker Compose (v2)

### Запуск

```docker compose up --build```

```curl -v http://localhost``` либо открыть в бразуере ссылку http://localhost
